# Pterodactyl + Bedrock Server Installer

This repo contains everything needed to deploy a fully managed **Minecraft Bedrock Dedicated Server** on a fresh Ubuntu 22.04 VPS — all controlled from the Pterodactyl panel.

## Quick Start (VPS Terminal)
```bash
git clone https://github.com/yourname/pterodactyl-bedrock-installer.git

sudo ./install.sh
